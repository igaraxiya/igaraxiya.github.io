利用hexo快速搭建blog

一、基本组件安装

首先安装 Node.js 和 Git

完成后安装hexo

npm install -g hexo-cli

安装完成后找一个自己中意的目录

hexo init <name>
cd <name>
npm install

其中<name>是你想要存储所有文件的文件夹名，完成后文件夹的目录如下：

.
├── _config.yml
├── package.json
├── scaffolds
├── source
|   ├── _drafts
|   └── _posts
└── themes

随后运行

hexo g 
hexo s

之后就可以在http://localhost:4000/中查看网页

二、部署到GitHub

没有github则先注册一个

之后在主页点击左上角的new新建一个仓库

Repository name为username.github.io，选择Public，ADD a REDME file

随后进入仓库，点击Setting，向下滑动找到GitHub Pages，将source中的branch改为master

之后在创建的blog文件夹中右击空白处Git Bash Here，输入

ssh-keygen -t rsa -C "your e-mail"

创建SSH Key，如果有则可以跳过。

可以通过下面的方法检查有无SSH Key

cd ~/.ssh

随后进入github->右上头像->Settings->SSH and GPG keys->new SSH key

title填写生成SSH时的email

key则通过下面指令，讲显示的代码复制进去即可

cd ~/.ssh
cat id_rsa.pub

输入

ssh -T git@github.com

如果显示如下内容即配置成功

Hi username! You've successfully authenticated, but GitHub does not
provide shell access.

最后打开创建的blog文件夹根目录下的_config.yml，修改最后面的内容为

# Deployment
## Docs: https://hexo.io/docs/deployment.html
deploy:
  type: git
  repo:
      github: git@github.com:<username>/<username>.github.io.git,master

username为github的用户名

最后执行

hexo g 
hexo d

就可一通过访问http://username.github.io 访问blog了
